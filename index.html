The creator

The Creator of the Microsoft computer was made by Bill gates. He was born on October 28, 1955 in Seattle, Washington. He created the first Microsoft computer called Microsoft Windows and was announced on November 10, 1983.

A new beginning

The first versions of Windows were actually just programs run from MS-DOS which then took over the screen and launched an application called Program Manager. But then we go to Windows 95 which was gonna get things bigger.Windows 95 merged Microsoft's formerly separate MS-DOS and Windows products.It featured significant improvements over its predecessor, Windows 3.1, most notably in the graphical user interface.

Windows CE 1.0

In 1996 microsoft created Windows CE 1.0, a new OS designed for devices with low memory and other constraints, such as personal digital assistants.

Windows XP

On October 25, 2001, Microsoft released Windows XP, Microsoft called the XP release its most important product since Windows 95. Along with a redesigned look and feel to the user interface, the new operating system was built on the Windows 2000 kernel, giving users a more stable and reliable environment than previous versions of Windows.

The Xbox

The Xbox is a home video game console and the first installment in the Xbox series of consoles manufactured by Microsoft. It was released on November 15, 2001 in North America, followed by Australia, Europe and Japan in 2002.

Windows 7

was officially released to the public. Windows 7's focus was on refining Vista with ease of use features and performance enhancements, rather than a large reworking of Windows.
